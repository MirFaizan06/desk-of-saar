## Instructions for Site Admin

When you add new resources (images, music, etc.), update this file using the format above to maintain proper attribution.

Example for adding a music track:
```
[MUSIC] Calm Focus Music
Author: Artist Name
Source: YouTube Audio Library
URL: https://youtube.com/audiolibrary/music
License: Royalty-Free
---
```

After updating this file, the changes will automatically be available for download on the Credits page.
