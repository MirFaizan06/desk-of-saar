# Music Files for Reading Mode

This folder contains background music tracks that play during Reading Mode.

## Setup Instructions

1. Add 3 MP3 files to this directory with the following names:
   - `track1.mp3` - Ambient Focus music
   - `track2.mp3` - Peaceful Reading music
   - `track3.mp3` - Calm Study music

2. Make sure the files are:
   - In MP3 format
   - Royalty-free or you have the rights to use them
   - Instrumental/ambient (good for focus and reading)
   - Moderate volume (they'll be controlled by the user)

## Recommended Music Sources

You can find royalty-free music at:
- [YouTube Audio Library](https://www.youtube.com/audiolibrary)
- [Incompetech](https://incompetech.com/music/)
- [Free Music Archive](https://freemusicarchive.org/)
- [Bensound](https://www.bensound.com/)
- [Purple Planet](https://www.purple-planet.com/)

## Music Genres That Work Well

- Ambient
- Classical
- Lo-fi
- Nature sounds
- Meditation music
- Study beats

## File Example Names

Once you add your files, you should have:
```
public/music/
├── track1.mp3
├── track2.mp3
├── track3.mp3
└── README.md (this file)
```

## Notes

- The music player in Reading Mode will automatically loop through all tracks
- Users can control volume, play/pause, and skip tracks
- The music plays in the background while users read PDFs
